Method: 1
You will need to run this project on a server, like Mamp ie. localHost

Method: 2
You will need to have Node installed, open a CMD (command prompt) navigate into the parent folder,
where app.js is located, and run the command "node app.js", if you are successful you will see the message
"Server running at http://localhost:3000/", this means the node server is running.
to see the application, paste or type "http://localhost:3000/" into the address bar of a browser. 
to exit the server press "Ctrl c" or close the CMD, and the application will no longer be available for refresh.
